# ☕ Coffee Survey Analysis

> Academic project developed during the **Data Science Essentials with Python** course from the Cisco Networking Academy.

## 📖 Overview

This project explores coffee consumption habits through the analysis of survey data collected from coffee drinkers.

Using real-world survey datasets, the project investigates preferences related to coffee consumption, preparation methods, purchasing behavior, and demographic characteristics.

The analysis applies data science techniques to uncover patterns and trends within the coffee community.

---

## 🎯 Objectives

* Explore coffee consumption habits.
* Analyze preferences among coffee drinkers.
* Investigate purchasing and brewing behaviors.
* Identify trends across survey respondents.
* Practice data cleaning, transformation, and visualization techniques.

---

## 📊 Datasets

### Coffee Survey Full Dataset

Primary dataset containing survey responses from coffee consumers.

The dataset includes information related to:

* Coffee preferences
* Brewing methods
* Purchasing habits
* Consumption frequency
* Demographic characteristics

### Coffee Survey Results

Processed dataset containing cleaned and structured survey results used for analysis and visualization.

## 🔬 Analysis Performed

The project performs exploratory data analysis (EDA) to better understand the survey responses.

### Data Exploration

Initial exploration includes:

* Dataset dimensions
* Data types
* Missing values
* Variable distributions

### Data Cleaning

Data preparation steps may include:

* Handling missing values
* Removing invalid responses
* Standardizing categorical values
* Preparing data for visualization

### Survey Analysis

The project investigates topics such as:

* Most popular coffee types
* Preferred brewing methods
* Coffee consumption frequency
* Purchasing preferences
* Consumer behavior patterns
